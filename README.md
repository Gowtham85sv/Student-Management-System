# Student Management System

## Description

The Student Management System is a desktop-based GUI application developed using Python and the Tkinter library. This project is designed to simplify the management of student records through an easy-to-use graphical interface. It serves as a beginner-friendly project to understand GUI development, event handling, and basic data management in Python.

The application allows users to enter student details such as Name, Roll Number, Department, and Marks. Users can add new student records, search for students using their roll number, delete existing records, display all stored records, and clear input fields with simple button clicks.

## Features

• Add new student records.
• Search students by Roll Number.
• Delete selected student records.
• Display all stored student information.
• Clear all input fields.
• Simple and user-friendly graphical interface.

## Purpose

This project was created as a beginner-level Python GUI application to improve programming skills and gain hands-on experience in developing desktop applications using Tkinter. It provides a strong foundation for building more advanced management systems in the future.
